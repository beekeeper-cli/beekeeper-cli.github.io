retry

xports a function which uses exponential backoff to retry a throttled asynchronous function multiple times.

Examples
const retry = require('./retry');

one line

(await retry(() => myThrottledFunction()))();

multi-line

(await retry(() => {
   myThrottledFunction()
 })
)();
